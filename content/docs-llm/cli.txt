SurrealDB CLI tool overview: Quickstart with `surreal start --user root --pass secret` (in-memory, serves at 127.0.0.1:8000). Interactive shell via `surreal sql --username root --password secret --pretty` (defaults to NS=main, DB=main). Use `--namespace`, `--database` to customize.
