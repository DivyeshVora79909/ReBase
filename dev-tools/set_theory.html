All definitions, symbols, and axioms — compact & exam-ready

1. Core Symbols & Basics

  • ∈ — element of   ∉ — not element of
  • ∅ — empty set   |∅| = 0
  • ⊆ — subset (all elements of A are in B)
    ⊂ — proper subset (strict, not equal)
  • = — set equality
    A = B ⇔ ∀x (x ∈ A ⇔ x ∈ B) (Axiom of Extension)
Note: Sets are unordered and ignore duplicates: {1,2} = {2,1,2}.

2. Set Operations

  • ∪ — union A ∪ B = { x | x ∈ A ∨ x ∈ B }
  • ∩ — intersection A ∩ B = { x | x ∈ A ∧ x ∈ B }
  • \ or − — difference A \ B = { x | x ∈ A ∧ x ∉ B }
  • ∁ or ^c — complement A^c = U \ A (relative to a fixed universe U)

Generalised (indexed) operations

  • ⋃i∈I Ai = { x | ∃i ∈ I : x ∈ Ai }
  • ⋂i∈I Ai = { x | ∀i ∈ I : x ∈ Ai }

3. Advanced Constructions

  • × — Cartesian product
    X × Y = { (x,y) | x ∈ X, y ∈ Y }
  • 𝒫(X) — power set
    𝒫(X) = { S | S ⊆ X }
    |𝒫(X)| = 2|X| (finite)
  • ⊔ or ⊎ — coproduct (disjoint union / tagged union)
    A ⊔ B = (A × {0}) ∪ (B × {1})
  • Polymorphism: a field maps to A ⊔ B ⊔ C (exactly one, tagged)
Cartesian vs Power: X × Y is the grid of pairs. 𝒫(X × Y) is the set of all possible relations.

4. Relations

A relation is a subset of a Cartesian product: R ⊆ X × Y.

  • Dom(R) = { x | ∃y, (x,y) ∈ R } — domain (left set)
  • Cod(R) = Y — codomain (right set, declared)
  • Range(R) = { y | ∃x, (x,y) ∈ R } — image (used subset of codomain)
  • R−1 = { (y,x) | (x,y) ∈ R } — inverse relation
  • R(x) = { y | (x,y) ∈ R } — the set of all y related to x
Preimage: R−1(y) = { x | (x,y) ∈ R } → counting this gives you cardinality (1, N, 0).

5. Functions (Mappings)

  • f: X → Y — total function
    ∀x ∈ X, |f(x)| = 1 (each x maps to exactly one y)
  • f: X ⇀ Y — partial function
    ∀x ∈ X, |f(x)| ∈ {0,1} (optional 0..1)
  • f: X → 𝒫fin(Y) — set-valued (list / multi-value)
    each x maps to a set of y's

Properties of functions

  • Injective (1‑to‑1): f(x₁)=f(x₂) ⇒ x₁=x₂
  • Surjective (onto): ∀y ∈ Y, ∃x : f(x)=y
  • Bijection: injective + surjective → strict 1:1, inverse is also a function

6. Equivalence & Order Relations

A relation R ⊆ X × X (on a single set) can be:

  • Reflexive: ∀x, (x,x) ∈ R
  • Symmetric: (x,y)∈R ⇒ (y,x)∈R
  • Transitive: (x,y)∈R ∧ (y,z)∈R ⇒ (x,z)∈R

Equivalence relation (~)

If R is reflexive, symmetric, and transitive → it partitions X into equivalence classes [x] = { y | x ~ y }.

The set of all classes is the quotient set: X / \sim.

Order relations

  • Partial order (≤): reflexive, transitive, antisymmetric (xRy ∧ yRx ⇒ x=y).
  • Total order: every pair is comparable (xRy or yRx).

7. Cardinality (Sizes)

  • |S| — cardinality (number of elements)
  • |\emptyset| = 0
  • Countably infinite: |\mathbb{N}| = \aleph_0 (natural numbers)
  • Uncountably infinite: |\mathbb{R}| = 2^{\aleph_0} (real numbers)
  • |X| < |\mathcal{P}(X)|   Cantor's theorem: power set is always strictly bigger.
Key insight: There are infinitely many infinities. The continuum hypothesis asks if \aleph_1 = 2^{\aleph_0}.

8. Tuples & n‑ary Products

Ordered pair (Kuratowski): (a,b) = { {a}, {a,b} }

Generalised n‑tuple:

X₁ × X₂ × … × Xn = { (x₁, x₂, …, xn) | xi ∈ Xi }

Used for composite keys, record types, and relation schemas.

9. Axioms & Important Caveats

  • Axiom of Extension: sets are equal iff they have the same elements.
  • Axiom of Foundation: no set contains itself (A ∉ A).
  • Russell's Paradox: there is no universal set U that contains everything.
    The "universe" in practice is a class or a fixed domain of discourse.
  • Complement (A^c) is always relative to a specific, fixed universal set U — never a true "set of all sets".
⚠️ Important: In formal set theory, U is a class, not a set. Complements are relative.

📌 Quick Reference — The Big Picture

  • Relation R ⊆ X × Y — any set of pairs.
  • Total function f: X → Y — every x maps to exactly one y. (Child → parent FK)
  • Partial function f: X ⇀ Y — at most one y. (Optional FK, 0..1)
  • Polymorphism T → (A ⊔ B ⊔ C) — tagged union / coproduct.
  • Set-valued f: X → 𝒫fin(Y) — each x maps to a set of y's (1:N via a list).
  • Cardinality check count the preimage: |f^{-1}(y)| gives you N:1, 1:1, etc.
